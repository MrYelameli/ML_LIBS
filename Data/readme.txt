1) all_rock_data.txt

It is a complete dataset of 10 rocks (1309x1024)

2) train_data.txt and test_data.txt

These are the derived dataset from all_rock_data.txt file with the ratio of 0.8 for training dataset and 0.2 for test dataset

3) Norm_all_rock_data.txt

This is a normalized dataset of all_rock_data.txt

4) PCA_teD.txt and PCA_trD.txt

A point to be noted, PCA is applied on Norm_all_rock_data.txt and then that dataset is divided into 0.8 for train and 0.2 for test dataset.
This is a direct input to main_old.m program. This dataset is used in ocean conference.
